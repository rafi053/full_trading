## 🔌 חיבור ל-Backend

הפרויקט מחובר ל-Backend שלך דרך:

```env
VITE_API_URL=http://localhost:3000
```

### נקודות API שהפרונט משתמש בהן:

```
Bots:
  GET    /api/bots
  POST   /api/bots
  PUT    /api/bots/:id
  DELETE /api/bots/:id
  POST   /api/bots/:id/start
  POST   /api/bots/:id/stop

Positions:
  GET    /api/positions
  POST   /api/positions/:id/close

Trades:
  GET    /api/trades
```
