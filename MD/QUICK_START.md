# ⚡ Quick Start - התחלה מהירה

## התקנה מהירה ב-3 שלבים

### 1️⃣ התקן תלויות

```bash
cd crypto-trading-ui
npm install
```

### 2️⃣ הגדר את כתובת ה-Backend

ערוך את קובץ `.env`:

```env
VITE_API_URL=http://localhost:3000
```

### 3️⃣ הפעל את הפרויקט

```bash
npm run dev
```

🎉 **זהו!** האתר פועל ב-`http://localhost:5173`

## דוגמת שימוש מהירה

### יצירת בוט

1. לך ל-"Trading Bots"
2. לחץ "Create New Bot"
3. מלא:
   - **שם**: "My First Bot"
   - **סוג**: Trend Long
   - **בורסה**: Binance
   - **סימול**: BTC/USDT
   - **מינוף**: 5x
   - **גודל**: 100 USDT
4. שמור

### הפעלת בוט

- לחץ על כפתור **"Start"** ליד הבוט
- הבוט יתחיל לסחור אוטומטית!

---

## נתקעת?

### הבוטים לא נטענים?

```bash
# בדוק שה-Backend רץ
curl http://localhost:3000/api/bots
```

### שגיאת CORS?

ודא שבקובץ `src/corsConfig.ts` של ה-Backend יש:

```typescript
origin: ['http://localhost:5173']
```

### רוצה לעדכן נתונים?

כל הנתונים מתעדכנים אוטומטית כל 3-5 שניות!
